Skupper.Skupper

Collection version 1.8.3

  • Description

  • Plugin Index

  • Role Index

Description

Provides roles and modules to help managing Skupper sites

Author:

  • Fernando Giorgetti <fgiorgetti@gmail.com>

Supported ansible-core versions:

  • 2.15.0 or newer

  • Issue Tracker
  • Homepage
  • Repository (Sources)

Plugin Index

These are the plugins in the skupper.skupper collection:

Modules

  • links module – Update links based on provided links list

  • links_load module – Loads existing links to other sites

  • services module – Update services based on provided services list

  • services_load module – Loads existing services and targets

  • site_load module – Loads site information as ansible facts into the respective host

Role Index

These are the roles in the skupper.skupper collection:

  • skupper_cli_install role – Installs the Skupper CLI

  • skupper_init role – Initializes a Skupper site

  • skupper_link role – Maintains existing links on a site

  • skupper_option role – Provides common parameters to be used across the skupper.skupper roles.

  • skupper_service role – Maintains existing services on a site

  • skupper_setup role – Configures your entire Virtual Application Network (VAN)

  • skupper_teardown role – Deletes a Skupper site

  • skupper_token role – Creates a Skupper token that can be used by other sites to establish links

  • skupper_update role – Updates a Skupper site
