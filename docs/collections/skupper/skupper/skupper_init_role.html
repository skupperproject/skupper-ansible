skupper.skupper.skupper_init role – Initializes a Skupper site

Note

This role is part of the skupper.skupper collection (version 1.8.3).

It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it use: ansible-galaxy collection install skupper.skupper.

To use it in a playbook, specify: skupper.skupper.skupper_init.

Entry point main – Initializes a Skupper site

New in skupper.skupper 1.6.0

Synopsis

  • Initializes a Skupper site based on provided options and parameters.
    Some arguments are only used when skupper_option_platform is set to kubernetes
    and others when set to podman.
    These arguments are similar to the arguments accepted by the Skupper CLI, when
    the initialization is performed against a given platform (kubernetes or podman).
