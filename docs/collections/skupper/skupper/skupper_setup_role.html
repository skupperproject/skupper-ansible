skupper.skupper.skupper_setup role – Configures your entire Virtual Application Network (VAN)

Note

This role is part of the skupper.skupper collection (version 1.8.3).

It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it use: ansible-galaxy collection install skupper.skupper.

To use it in a playbook, specify: skupper.skupper.skupper_setup.

Entry point main – Configures your entire Virtual Application Network (VAN)

New in skupper.skupper 1.6.0

Synopsis

  • The skupper_setup role performs the entire Skupper network deployment,
    defining links and services across all sites.
    It combines the execution of all the following roles:
    * skupper.skupper.skupper_init
    * skupper.skupper.skupper_token
    * skupper.skupper.skupper_link
    * skupper.skupper.skupper_service
