skupper.skupper.skupper_update role – Updates a Skupper site

Note

This role is part of the skupper.skupper collection (version 1.6.0).

It is not included in ansible-core. To check whether it is installed, run ansible-galaxy collection list.

To install it use: ansible-galaxy collection install skupper.skupper.

To use it in a playbook, specify: skupper.skupper.skupper_update.

Entry point main – Updates a Skupper site

New in skupper.skupper 1.6.0

Synopsis

  • Updates a Skupper site based on provided skupper options. For more information see skupper.skupper.skupper_option role.
